# FUE Connect

FUE Connect is a mobile application developed as a graduation project at Future University in Egypt (FUE) to provide students with academic tools, communication features, opportunities, and campus services through a unified platform.

## Technologies Used
- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- Push Notifications

## Features
- User Authentication
- Student Profiles
- Posts & Community Interaction
- Live Chat
- Notifications
- Academic Calendar
- GPA Calculator
- Campus Map
- Jobs & Internship Opportunities
- Volunteer Opportunities
- Student Support Services
- Admin Dashboard

## Development Team
Developed by a team of 6 students as a graduation project at Future University in Egypt.
